مايكروسوفت تستحوذ على GitHub في صفقة قيمتها 7.5 مليار دولار 

 أعلنت شركة مايكروسوفت أمس عن استحواذها على على منصة GitHub لمشاركة التعليمات البرمجية والمشهورة لدى العديد من مطوري البرمجيات في صفقة قيمتها 7.5 مليار دولار .بحسب ما أعلنت عنه مايكروسوفت سيتولى نات فرايدمان نائب رئيس الشركة ومؤسس شركة Xamarin منصب المدير التنفيذي لـ GitHub بعد الاستحواذ، فيما سيصبح المدير التنفيذي السابق لمنصة GitHub كريس وانسترات مستشاراً لنائب رئيس مايكروسوفت للعمل على استراتيجيات جديدة لتطوير البرمجيات، حيث سيرفع التقارير إلى نائب الرئيس التنفيذي سكوت غوثري.إن حقيقة قيام مايكروسوفت بتعيين رئيس تنفيذي جديد لشركة GitHub هي علامة واضحة على أن نهج الشركة في دمج GitHub سيكون مماثلاً لنجاحها في العمل مع لينكدإن. وقالت مايكروسوفت في منشور عبر مدونتها: “ستحتفظ منصة GitHub بروح التطوير التي يقدمها المطورين وستعمل بشكل مستقل لتوفير منصة مفتوحة لجميع المطورين في جميع الصناعات، وسيستمر المطورون في استخدام لغات البرمجة والأدوات وأنظمة التشغيل التي يختارونها لمشاريعهم – وسيظل بإمكانهم نشر أكوادهم البرمجية في أي نظام تشغيل وأي سحابة تخزين وأي جهاز”.ومن جانبه قال وانسترات: “أنا فخور للغاية بما أنجزه GitHub ومجتمعنا خلال العقد الماضي و سعيد للغاية لأنني سأشارك مع مايكروسوفت لمساعدة GitHub في تحقيق المزيد من التقدم والتطوير وسوف يساعدنا على ذلك أدوات مايكروسوفت البرمجية وسحابتها العالمية لجعل GitHub أكثر قيمة للمطورين في كل مكان”.على الرغم من شعبية منصة GitHub بين مستخدمي الشركات والمطورين الأفراد والمشروعات مفتوحة المصدر،الإ أنها عانت من خسائر مالية في الفترة الأخيرة وفرصة الاستحواذ كانت أفضل من محاولة طرحها للاكتتاب، وقد فضلت إدارة GitHub بيع الشركة لمايكروسوفت بسبب إعجابهم بالرئيس التنفيذي ساتيا ناديلا. الجدير بالذكر أن منصة GitHub تأسست في عام 2008 ويقع مقرها في سان فرانسيسكو ووصل آخر تقييم لها  إلى 2 مليار دولار في عام 2015.تمنح عملية الاستحواذ  GitHub وسيلة للاستمرار في السوق حيث كانت تحاول لمدة تسعة أشهر العثور على مدير تنفيذي جديد كما لم تحقق أرباحاً بعد من خدمتها الشهيرة التي تسمح للمبرمجين بالمشاركة والتعاون في عملهم. وستساعد هذه الصفقة شركة مايكروسوفت التي تعتمد بشكل متزايد على البرمجيات مفتوحة المصدر لإضافة أدوات برمجة جديدة وربطها بمنصة أصبحت جزءًا أساسيًا من الطريقة التي تتبعها مايكروسوفت في كتابة برمجياتها الخاصة.تعتبر GitHub أداة أساسية للمبرمجين، كما يستخدممها العديد من المؤسسات العامة بما في ذلك شركة مايكروسوفت وألفابت لتخزين مصادر التعليمات البرمجية التابعة لها، وتعتبر أيضًا مجتمعًا مركزيًا لكل أنواع المطورين. عانت GitHub في الفترة الأخيرة من خسائر كبيرة حيث خسرت 66 مليون دولار على مدى تسعة أشهر في عام 2016.والجدير بالذكر أنه في أغسطس الماضي أعلنت GitHub أنها تبحث عن رئيس تنفيذي ليحل محل كريس وانسترات أحد المؤسسين المشاركين للشركة. وفي غضون ذلك انضم رئيس الشؤون التجارية في GitHub خوليو أفالوس إلى مجلس إدارة الشركة واستحوذ على الكثير من مسؤوليات القيادة اليومية للشركة.يستضيف موقع GitHub ما يصل إلى 27 مليون مطور برمجيات يعملون على 80 مليون مخزن لمصادر التعليمات البرمجية بلغات برمجية مختلفة، وشركة مايكروسوفت التي كانت في السابق معارضة لهذا النوع من تطوير البرامج مفتوحة المصدر أصبحت الآن واحدة من أكبر المساهمين في GitHub، حيث يعمل المدير التنفيذي لشركة مايكروسوفت ساتيا ناديلا على إبعاد الشركة عن الاعتماد الكامل على نظام التشغيل ويندوز مع المزيد من التطوير الداخلي بالاعتماد على لينكس، لذلك فإن الشركة تحتاج إلى طرق جديدة للتواصل مع مجتمع المطورين على نطاق واسع.
