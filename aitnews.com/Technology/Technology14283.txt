شركة مايكروسوفت تطرح إصدار Windows Server 2003 R2 

 بدأت شركة مايكروسوفت في الأسبوع الماضي طرح إصدار Windows Server 2003 R2، والذي طرحته للتصنيع في أوائل ديسمبر الماضي. ويعد هذا الإصدار تحديثًا مؤقتًا لنظام تشغيل الخادم ليفصل بين Server 2003 SP1 وLonghorn Server المقرر طرحه في عام 2007.ولا تعد شركة مايكروسوفت أولى الشركات التي تطرح R2 على الخوادم الخاصة بها، فلقد طرحتها شركة إتش بي وديل وآي بي إم. وتعتزم شركة مايكروسوفت استبدال إصدارات Windows Server 2003 الموجودة حاليًا في مراكز البيع بالإصدار الجديد.
