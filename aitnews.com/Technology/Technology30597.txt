مايكروسوفت تُطلق مُحرر الشيفرات البرمجية Visual Studio Code 

 أعلنت شركة مايكروسوفت اليوم خلال مؤتمر Build عن إطلاق مُحرر جديد للأكواد البرمجية حمل اسم Visual Studio Code وهو متوفر مجانًا للتحميل لأنظمة ويندوز، ماك بالإضافة إلى لينكس.المُحرر الجديد مُتخصص بكتابة وتعديل الشيفرات البرمجية Codes مع الكثير من الميزات من أهمها ميّزة الإكمال التلقائي الذكي IntelliSense التي تقوم بعرض اقتراحات لإكمال الشيفرة بشكل آلي، وبالتالي لا يحتاج المُستخدم لكتابة كل شيء بشكل يدوي.ويُمكن للمستخدمين التوجه إلى الرابط https://code.visualstudio.com لتحميل المُحرر الذي يدعم لغتي Asp.net وNode.js لبناء تطبيقات ويب سحابية. ومن المتوقع أن تُطلق الشركة لاحقًا النسخة المُرشّحة Release Candidate من إطار عمل فيجوال ستوديو Visual Studio 2015 الذي يدعم ماك ولينكس بالإضافة إلى ويندوز.
