كاسبرسكي لاب تطلق حلها الأمني للبيئات الافتراضية 

 أعلنت كاسبرسكي لاب عن إطلاق أحدث تقنياتها الأمنية الخاصة بقطاع الأعمال وهو Kaspersky Security for Virtualization- الذي صمم من الصفر من قبل فريق خبراء الأمن في كاسبرسكي لاب- ويركز على حماية البنى التحتية المؤسساتية النامية من البرمجيات الخبيثة والمجرمين الالكترونيين.ويعمل Kaspersky Security for Virtualization كأداة أمنية افتراضية تتكامل مع VMware vShield™ Endpoint لتؤمن حماية من البرمجيات الخبيثة بدون وكيل، معتمدة على المحرك المكافح للبرمجيات الخبيثة المطور من كاسبرسكي لاب.ويتميز Kaspersky Security for Virtualization بتجنب التعقيدات غير الضرورية واستنزاف الموارد التي تتصف بها البرامج الأمنية التقليدية، التي بدورها تكرس التقنية المطلوبة لخدمة البيئات الافتراضية. ويحظى مستخدمو حلول كاسبرسكي لاب بأداة إدارية موحدة لبيئاتهم الحاسوبية الافتراضية، الفعلية والمتنقلة. إلى جانب التوافقية الفورية مع منتجات كاسبرسكي لاب للنقاط النهائية، تتكامل Kaspersky Security for Virtualization بسلاسة مع VMware vShield™ Endpoint Security وتؤمن حماية مباشرة لمراكز البيانات الافتراضية، الخوادم الافتراضية والحواسب المكتبية. فباستخدام VMware vShield™ Endpoint، يقوم Kaspersky Security for Virtualization بتشخيص جميع الآلات الافتراضية ضمن الشبكة، ناقلا الملفات إلى عميل كاسبرسكي لاب للبحث عن الفيروسات بنظام الزمن الفعلي.وقال بيوتر ميركولوف، مدير شؤون المنتجات لكاسبرسكي لاب، إن تكامل منتجات كاسبرسكي مع VMware يمنح العملاء إمكانية استخدام تقنيات كاسبرسكي لاب الرائدة في بنى VMware لمؤسساتهم. وأضاف، أن الشركة تسعى إلى تطوير فعالية أكبر مع VMware API، إلى جانب القدرة على عزل أية آلات افتراضية مصابة قبل انتشار البرمجيات الخبيثة في الشبكة.و من خلال التكامل مع VMware vShield™ Endpoint، يطبق منتج Kaspersky Security for Virtualization وظائفه المكافحة للبرمجيات الخبيثة على جهاز مركزي افتراضي، وهذا يزيل ضرورة استهلاك الموارد بشكل متكرر لكل آلة افتراضية على حده. إذ أنه بإدخار طاقات البيئة الافتراضية، Kaspersky Security for Virtualization يحسّن من الأداء ويقلل من الموارد المنفقة على عتاد الحاسوب واستهلاك الكهرباء ويؤمن تقارير مفصلة حول المهام الأمنية للتحقق من تلبية المتطلبات. إضافة إلى ذلك، عادة ما تقتني المؤسسات خليطا من البيئات الافتراضية والفعلية والمتنقلة، غير أن مديري النظام لا يحبذون التحكم بالبرامج متعددة المنصات من أدوات تحكم متعددة.وبإمكان مسؤولي تقنية المعلومات توفير السياسات الأمنية المعدلة لمجموعة الآلات الافتراضية المختارة بسهولة شأنها شأن الآلات الفعلية. إن الإدارة الموحدة لجميع البيئات الحاسوبية تؤمن مزيجا فريدا من الإدارة المباشرة مع الحماية من الجيل الجديد.ويضمن استخدام Kaspersky Security for Virtualization حماية جميع الآلات الافتراضية عبر نفس قاعدة بيانات البرمجيات الخبيثة المركزية والتي يتم تحديثها باستمرار مع ظهور تهديدات جديدة. هذا النموذج من الحماية يكفل تزويد جميع الآلات الافتراضية بالمستوى نفسه من الحماية بغض النظر عن تاريخ إنشاء الآلة الافتراضية أو المدة التي يتم تشغيلها فيها. ولن تكون هناك “تذبذبات” قصيرة في عمل الحل على الآلات الافتراضية التي لم تستخدم لفترات طويلة من الزمن كما لن يحدث تدن في الأداء لدى تحديث الآلات الافتراضية.ويوفر Kaspersky Security for Virtualization حماية بدون عميل للآلات الافتراضية التي تشغل على جميع أنظمة تشغيل Windows، بما فيها Widows 7، Windows Vista، Windows XP، Windows Server 2008، Windows Server 2008 R2 و Windows Server 2003.ويندرج Kaspersky Security for Virtualization ضمن الحلول Kaspersky Targeted Security ابتداء من 10 أبريل 2012.
