نظام تشغيل جديد يدعم ربط تقنية الحوسبة السحابية للشبكات 

وفرت "مايكروسوفت" نظام التشغيل Windows Server 2012 بالمجان عبر موقعها الرسمي حاليا، ويمنح المستخدم فرصة تشغيل النظام الجديد لمدة 180 يوما وبكامل الخصائص وجميع مزاياه، والربط بعدد لا محدود من الأجهزة في شبكة محلية. ويساعد نظام التشغيل Windows Server 2012 المؤسسات والشركات بمختلف مستوياتها على الاتصال بأمان عبر أماكن العمل وتنظيم الشبكات البينية وتعريفها وتنظيم أداء ومهام الأجهزة المتصلة بالشبكة الداخلية، ويساعد متخصصي تكنولوجيا المعلومات لتحقيق جميع الاحتياجات التجارية أسرع وأكثر كفاءة عن الإصدارات السابقة.
 ومن أبرز مهام نظام التشغيل Windows Server 2012 أنه يتيح إمكانية إعادة تعريف فئة الخادم، كما توفر تقديم المئات من الميزات الجديدة والتحسينات التي تدعم خصائص البروتوكولات الافتراضية، والربط الشبكي المتعدد، وتنظيم أدوات التخزين لجميع مهام المستخدم، والربط الاحترافي مع الحوسبة السحابية المتوافق مع نظم الأجهزة الذكية، وتنشيط التشغيل الآلي، وأكثر من ذلك. ونظام التشغيل Windows Server 2012 يساعد على تنفيذ جميع عمليات تكنولوجيا المعلومات الخاصة بالمنشأة بأقل التكاليف وتقديم مستوى عالي الكفاءة يرفع من قيمة الأعمال.
 ووصفت الشركة الخادم الجديد بأنه ذو أداء احترافي لنظم تشغيل السحابة Cloud OS الذي يزود العملاء بنظم حديثة تواكب النهضة الهائلة لخدمات السحابة المستمرة، والأجهزة المتصلة والبيانات الضخمة، وكذلك في التطبيقات وخدمات الحوسبة السحابية الخاصة بالشركة أو المرتبطة بها لتقديم نظام تشغيل السحابة المعتمد على خادم ويندوز و Windows Azure لتساعد المستخدمين على إنجاز مراكز بيانات بلا قيود.
 وهذا الإجراء من شركة مايكروسوفت الذي قدمت خلاله نظام تشغيل ويندوز الخادم 2012 الجديد ليتمكن صانعو القرارات الاقتصادية ومحترفو تكنولوجيا المعلومات في المنشآت الصغيرة والكبيرة من الاختبار العملي وتجربة نظام السحابة المركب الذي يعتمد على خادم ويندوز الجديد Windows Server 2012 وعلى منصات Windows Azure.
 و"مايكروسوفت" دعمت ببناء خادمWindows Server 2012 التوافق مع تطبيقات الحوسبة السحابية، لتوظيف خبراتها الطويلة في تشغيل مراكز البيانات الدولية التي ترتكز على مئات الآلاف من أجهزة الخوادم لتقديم أكثر من 200 من خدمات الحوسبة السحابية، ويقوم الخادم الجديد بتوسيع تعريف نظام تشغيل الخوادم ليدمج تقنيات تقليدية منفصلة مثل التخزين المتقدم، والتشبيك، وقدرات الخوادم الافتراضية والمهام الأوتوماتيكية.
 ومن خلال المئات من الخصائص الجديدة يستطيع العملاء تحقيق نقلة هائلة في السرعة والمستوى وقدرة التطبيقات ومراكز البيانات الخاصة بهم. ومع جمع الخادم الجديد مع Windows Azure و System Center يتم دعم العملاء لإدارة وتقديم التطبيقات والخدمات عبر أنظمة السحابة الخاصة والمستضافة والعامة.
 ويستطيع العملاء استخدام مهاراتهم الحالية في إدارة الأنظمة، وتنصيب مختلف أنواع التطبيقات وتطويرها، وقواعد البيانات، والتثبت من الهوية، والبيئات الافتراضية للاستفادة من مميزات Windows Server 2012، والتجهيزات الملحقة كالطابعات والماسح الضوئي ودعم النسخ الاحتياطي، وغيرها. ويمكن لمن يرغب في تحميل نظام التشغيل للخادم من خلال الدخول إلى موقع "مايكروسوفت" وبالمجان لفترة ستة أشهر من بداية تشغيله.
