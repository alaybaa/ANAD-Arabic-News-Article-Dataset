بالصور ... الأهلي يغادر إلى العين بقائمة تضم اثنين وعشرين لاعباً 

 غادرت بعثة الفريق الكروي الأول في النادي الأهلي مطار الملك عبدالعزيز الدولي بجدة متوجهة إلى مدينة العين في دولة الإمارات لإقامة معسكر إعدادي هناك يستمر حتى السادس عشر من شهر يناير الجاري ، وذلك استغلالاً لفترة التوقف الحالية وتحقيقاً لأقصى مراحل الإعداد الفني والبدني بما يضمن حضوراً مميزاً للفريق بعد عودة المنافسات خصوصاً وأنه سيكون بانتظار نجوم الملكي مواصلة مشوارهم في دوري عبداللطيف جميل للمحترفين الذي يتصدره الفريق مع نهاية الدور الأول بالإضافة إلى نهائي كأس سمو ولي العهد ودوري المجموعات في دوري أبطال آسيا وكذلك مسابقة كأس خادم الحرمين الشريفين . وكانت البعثة المغادرة ظهر اليوم برئاسة الأستاذ باسم أبوداود المدير التنفيذي لكرة القدم ضمت مدرب الفريق كريستيان جروس وجهازه الفني المساعد المكون من : لورنت هاجست ( مساعد المدرب ) كونل ( مساعد المدرب ) اسماعيل مسعود ( مدرب الحراس ) شيخ بوزيان ( مدرب اللياقة ) مشاري الغامدي ( مترجم الفريق ) ، والجهاز الاداري المكون من : موسى المحياني ( مدير الكرة ) محمد شليه ( إداري الفريق ) محمد الزيد ( منسق الفريق ) فهد الزهراني ( المنسق الإعلامي ) ريان أبوهبرة ( مصور المركز الإعلامي ) والجهاز الطبي المكون من : وليد عبدالستار ( طبيب الفريق ) اسلام عباس ( اخصائي العلاج الطبيعي ) محمد البربري ( اخصائي العلاج الطبيعي ) ابراهيم اسماعيل ( اخصائي تأهيل ) علي يوسف ( مدلك ) محمود بدر ( مدلك ) بالإضافة إلى اثنين وعشرين لاعباً وهم : مهند عسيري - ياسر المسيليم - وليد باخشوين - تيسير الجاسم - اسلام سراج - أسامه هوساوي - سلمان المؤشر - أمير كردي - عبدالله المعيوف - عبدالله المطيري - محمد عبدالشافي - عقيل بلغيث - محمد آل فتيل - معتز هوساوي - حسين المقهوي - عمر السومة - كامل الموسى - محمد أمان - منصور الحربي - باسم العطالله - فهد حمد - ريان الموسى .
