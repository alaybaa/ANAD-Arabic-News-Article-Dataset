أولمبي الهلال يكسب لقاء الوحدة 

 تغلب فريق أولمبي الهلال لكرة القدم عصر اليوم الجمعة على مضيفه فريق "الوحدة" بهدفين مقابل هدف، وذلك في لقاء أقيم على ملعب نادي "الوحدة" بمكة المكرمة، ضمن لقاءات الجولة الأولى من مسابقة كأس دوري الأمير فيصل بن فهد للموسم الرياضي 2016-2017، حيث سجل هدفي الهلال الثنائي "حسين سالم" و"عبدالعزيز الجمعان". كما أقيمت اليوم خمسة مباريات ضمن مباريات الجولة الأولى من دوري كأس الأمير فيصل بن فهد وجاءت نتائجها كالتالي: هجر - القادسية ( 0 - 1 )الفتح - الفيصلي ( 0 - 0 )الخليج - الأهلي ( 1 - 0 )النصر - التعاون ( 4 - 1 )أحد - الاتحاد ( 1 - 0 ) وجاء الترتيب على النحو التالي: 1 - النصر ( 3 نقاط )2 - الهلال ( 3 نقاط )3 - أحد ( 3 نقاط )4 - الخليج ( 3 نقاط )5 - القادسية ( 3 نقاط )6 - الجيل ( نقطة واحدة )7 - الشباب ( نقطة واحدة )8 - الفتح ( نقطة واحدة )9 - الفيصلي ( نقطة واحدة )10 - الوحدة ( بدون أي نقطة )11 - الأهلي ( بدون أي نقطة )12 - الاتحاد ( بدون أي نقطة )13 - هجر ( بدون أي نقطة )14 - التعاون ( بدون أي نقطة )
