الإنتربول السعودي يحذر من مواقع إلكترونية لبيع تذاكر أولمبياد لندن 

 حذر الانتربول السعودي بناءً على معلومات من نظيره في المملكة المتحدة عن بعض المواقع الاحتيالية التي تعرض تذاكر للألعاب الأولمبية 2012م المقامة حالياً في لندن والمواقع المشبوهة التالية : ( www.euroteam.net ) ( www.tixnet.com ) ( www.olympicticket.info ) (www.euroteam.info ) ( www.euroteamtickets.com ) ( www.2012tickets.org ) ( www.londonsummergames.net ) (www.londonsummergames.org ) ( www.euroteam.travel ) ( www.worldticketservice.net ) وأوضح أن الطريقة الصحيحة لشراء التذاكر تتم فقط عن طريق الموقع الرسمي (www.tickets.london2012.com) أو عن طريق الجهات الرسمية بالمملكة العربية السعودية.
