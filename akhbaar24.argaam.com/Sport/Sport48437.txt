تبدأ بمواجهات نارية.. إعلان نتائج قرعة دوري أبطال أوروبا 

 سُحبت مساء اليوم (الخميس) في إمارة موناكو الفرنسية قرعة دوري أبطال أوروبا 2017 / 2018 لكرة القدم. وأوقعت القرعة ريال مدريد مع بوروسيا دورتموند وتوتنهام وأبويل، فيما يواجه برشلونة يوفنتوس وأولمبياكوس وسبورتينغ لشبونة. وفيما يلي نتائج القرعة: المجموعة A : - بنفيكا - مان يونايتد - بازل - سيسكا موسكو المجموعة B : - بايرن ميونخ - سان جيرمان - أندرلخت - سيلتك المجموعة C : - تشيلسي - أتلتيكو مدريد - روما - كارباخ المجموعة D : - يوفنتوس - برشلونة - أولمبياكوس - سبورتينغ لشبونة. المجموعة E : - سبارتاك موسكو - إشبيلية - ليفربول - ماريبور المجموعة F : - شاختار - مان سيتي - نابولي - فينورد المجموعة G : - موناكو - بورتو - بيشكتاش - لايبزيغ المجموعة H : - ريال مدريد - دورتموند - توتنهام - أبويل.
