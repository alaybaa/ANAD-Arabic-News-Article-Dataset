انفوغرافيا .. بوفون حامي عرين تشكيلة أوروبا وميسي يقود الهجوم 

 يحمي الدولي الايطالي واسطورة حراسة المرمى جيجي بوفون تشكيلة الاسبوع في دوري ابطال اوروبا والتي شهدت خط هجومي ناري بقيادة الارجنتيني ليونيل ميسي صاحب الهاتريك في مباراة فريقه يوم أمس امام مانشستر سيتي والتي انتهت برباعية نظيفة للكتلان امام مدربهم السابق بيب جوارديولا. وجاء تشكيلة الاسبوع من دوري أبطال أوروبا على النحو التالي : حراسة المرمى : بوفون ( يوفنتوس ). في خط الدفاع : لايون ( بورتو ) خوانفران ( أتلتيكو مدريد ) ماريانو ( أشبيلية ) في خط الوسط : شامبرلين ( ارسنال ) أوزيل ( ارسنال ) انيستتا ( برشلونة ) روبين ( بايرن ميونخ ) في خط الهجوم : ميسي ( برشلونة ) كافاني ( باريس سان جيرمان ) مارتينيز ( نابولي )
