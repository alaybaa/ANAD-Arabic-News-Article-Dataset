كأس اودي: توتنهام يحرز اللقب بعد الفوز على البايرن بركلات الترجيح 

 احرز نادي توتنهام هوتسبير الانكليزي لقب كأس اودي الودية بعد تغلبه على نادي بايرن ميونيخ الالماني بركلات الترجيح بعد انتهاء الوقت الاصلي بين الطرفين بالتعادل 2 - 2. بايرن ميونيخ الذي فضل مدربه نيكو كوفاتش دخول المباراة بتشكيلة الناشئين واراح عدد كبير من النجوم وجد نفسه متاخرا بنتيجة 2 - 0 بهدفي اريك لاميلا د 18 وكريستيان اريكسن د 59 ولكن المهاجم الشاب يان ارب اعاد الفريق البافاري الى اجواء المنافسة مقلصا الفارق 2 - 1 د 61. ومع دخول اللاعبين الاساسيين ضغط البايرن الى الامام واستطاع تسجيل هدف التعادل عبر الكندي الفونسو ديفيس د 81 لتنتهي المباراة بالتعادل 2 - 2 ويحتكم الفريقان الى ركلات الجزاء. وفي ركلات الجزاء سجل توتنهام 6 ركلات واضاع واحدة بينما سجل البايرن 5 ركلات واضاع اثنتان ليحرز السبيرز اللقب للمرة الاولى في تاريخهم. توتنهام : الدرفيلد ( سجل ) – اريكسن ( اضاع ) – هاري كاين ( سجل ) – سون هيونغ مين ( سجل ) – روليس ( سجل ) – سكيب ( سجل ) – تانغانغا ( سجل ) . بايرن ميونيخ : الابا ( اضاع ) – تياغو الكانتارا ( سجل ) - توماس مولر ( سجل ) – ريناتو سانشيز ( سجل ) – ليفاندوفسكي ( سجل ) – ساربيت سينغ ( سجل) - بواتنغ ( اضاع ) .
