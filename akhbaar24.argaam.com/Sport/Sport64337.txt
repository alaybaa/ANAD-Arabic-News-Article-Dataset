فريق ناشئين الهلال يكسب لقاء الرائد 

 تغلب فريق ناشئين الهلال لكرة القدم عصر اليوم الخميس على ضيفه فريق "الرائد" بخمسة أهداف دون رد، تناوب على تسجيلها الخماسي (رياض الغامدي، خالد ربيع، سعد الناصر، محمد الناصر وأحمد دغريري)، وذلك في لقاء أقيم على الملعب الرئيس بمقر النادي، ضمن لقاءات الجولة الأولى من مسابقة كأس الاتحاد السعودي لكرة القدم لدرجة الناشئين. كما أقيمت اليوم خمسة مباريات ضمن مباريات الجولة الأولى من كأس الاتحاد السعودي لدرجة الناشئين وجاءت نتائجها كالتالي: القادسية - عرعر ( 0 - 0 )الشباب - هجر ( 2 - 1 )النصر - الفتح ( 0 - 0 )الاتحاد - التعاون ( 4 - 0 )الأهلي - الاتفاق ( 5 - 0 ) وجاء الترتيب على النحو التالي: المجموعة الأولى : 1 - الأهلي ( 3 نقاط )2 - الهلال ( 3 نقاط )3 - الاتفاق ( بدون أي نقطة )4 - الرائد ( بدون أي نقطة ) المجموعة الثانية : 1 - الاتحاد ( 3 نقاط )2 - الفتح ( نقطة واحدة )3 - النصر ( نقطة واحدة )4 - التعاون ( بدون أي نقطة ) المجموعة الثالثة : 1 - الشباب ( 3 نقاط )2 - القادسية ( نقطة واحدة )3 - عرعر ( نقطة واحدة )4 - هجر ( بدون أي نقطة )
