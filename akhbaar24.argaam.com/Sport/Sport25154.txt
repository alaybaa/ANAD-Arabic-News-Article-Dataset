فريق شباب الهلال يتعادل مع القادسية 

 تعادل فريق شباب الهلال لكرة القدم عصر اليوم السبت مع ضيفه فريق شباب "القادسية" بدون أهداف، وذلك في لقاء أقيم على الملعب الرئيس بمقر النادي، ضمن لقاءات الجولة الأولى من دور المجموعات لبطولة كأس الاتحاد السعودي لكرة القدم لدرجة الشباب. كما أقيمت اليوم خمسة مباريات ضمن مباريات الجولة الأولى من كأس الاتحاد السعودي لدرجة الشباب وجاءت نتائجها كالتالي: الفتح - هجر ( 1 - 0 )الشباب - العدالة ( 2 - 4 )النصر - أحد ( 1 - 1 )الاتحاد - الرائد ( 4 - 1 )الأهلي - الوحدة ( 3 - 1 ) وجاء الترتيب على النحو التالي: المجموعة الأولى : 1 - الاتحاد ( 3 نقاط )2 - الأهلي ( 3 نقاط )3 - الوحدة ( بدون أي نقطة )4 - الرائد ( بدون أي نقطة ) المجموعة الثانية : 1 - الفتح ( 3 نقاط )2 - أحد ( نقطة واحدة )3 - النصر ( نقطة واحدة )4 - هجر ( بدون أي نقطة ) المجموعة الثالثة : 1 - العدالة ( 3 نقاط )2 - القادسية ( نقطة واحدة )3 - الهلال ( نقطة واحدة )4 - الشباب ( بدون أي نقطة )
