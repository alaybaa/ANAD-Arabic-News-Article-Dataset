التصنيف الشهري لأفضل الأندية العالمية : الهلال في الصدارة المحلية .. الإتحاد يتقدم محلياً .. الشباب ثالثاً .. والنصر يبقى رابعاً 

 تصدر فريق الهلال الأندية السعودية في تصنيف الإتحاد الدولي للتاريخ والإحصاء " IFFHS " الصادر اليوم الاربعاء 7 ذي القعدة 1432 / الموافق 5 اكتوبر القائمة الجديدة لافضل 400 نادي في العالم لــ شهر (سبتمبر ) حيث حل الفريق الهلالي في المركز الأول سعودياً والـ (156) عالمياً ، وجاء الإتحاد في المركز الثاني سعودياً والـ (214) عالمياً وثالثاً الفريق الشبابي في المركز الثالث سعودياً والـ (241 ) عالمياً ، وأخير الفريق النصراوي في المرتبة الرابعة سعودياً (380 ) عالمياً .  ترتيب الأندية السعودية للشهر الفائت :  الهلال (135) الشباب (225) الإتحاد (257) النصر (390)   ترتيب أفضل عشرة أندية عالمية :- 1- برشلونه ( أسبانيا ) 2-ريال مدريد ( أسبانيا ) 3-مانشستر يونايتد ( إنجلترا ) 4-بورتو ( البرتغال ) 5-شالكه ( ألمانيا ) 6-فيليز سارسفيلد ( الأرجنتين ) 7-بايرن ميونخ ( ألمانيا ) 8-انتر ميلان ( إيطاليا ) 9-مانشستر ستي ( إنجلترا ) 10- يونيفير سداد كاتلويكا ( تشيلي )
