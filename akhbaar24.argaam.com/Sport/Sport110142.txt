منتخب مصر يستدعي "حجازي" لمواجهتي أنجولا والجابون بتصفيات المونديال 

 أعلن الجهاز الفني لمنتخب مصر بقيادة كارلوس كيروش قائمة اللاعبين المختارين لعضوية الفريق استعدادا لخوض مباراتي أنجولا والجابون الشهر الحالي ضمن الجولتين الأخيرتين لتصفيات المجموعة الأفريقية السادسة لكأس العالم وضمت القائمة 26 لاعبا على النحو التالي: وقرر كيروش استدعاء لاعب الاتحاد أحمد حجازي، للوجود ضمن قائمة منتخب مصر لمواجهتي أنجولا والجابون. وجاءت قائمة منتخب مصر على النحو التالي: حراس المرمى:محمد الشناوي ( الأهلي )محمد أبو جبل ( الزمالك )محمد صبحي ( فاركو )محمود جاد ( انبي) الدفاع:أكرم توفيق ( الأهلي ) أحمد فتحي ( بيراميدز )أحمد حجازي ( الاتحاد )أحمد ياسين ( البنك الأهلي ) أيمن اشرف ( الأهلي )محمود علاء ( الزمالك ) أحمد فتوح ( الزمالك )محمد حمدي ( بيراميدز )محمود حمدي "الونش" ( الزمالك ) الوسط:محمد النني ( أرسنال الإنجليزي )حمدي فتحي ( الأهلي ) عمرو السولية ( الأهلي )إمام عاشور ( الزمالك )عبد الله السعيد ( بيراميدز )مصطفى فتحي ( الزمالك ) الهجوم:عمر مرموش ( شتوتجارت الألماني )محمد شريف ( الأهلي )أحمد ياسر ريان ( ألتاي التركي )مصطفى محمد ( جالاتا سراي التركي )مروان حمدي ( سموحة )محمد صلاح ( ليفربول الإنجليزي )محمد مجدي "أفشة" ( الأهلي )
