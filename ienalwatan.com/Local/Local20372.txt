أمانة العاصمة المقدسة ترفع كفاءة شبكة الأنفاق بمكة والمشاعر خلال موسم حج هذا العام 

سخرت أمانة العاصمة المقدسة ممثلةً في وكالة التشييد والصيانة كافة الإمكانات والموارد المتاحة لمتابعة تشغيل المرافق البلدية والأنفاق في مكة المكرمة والمشاعر المقدسة ورفع كفاءتها وجاهزيتها خلال موسم حج هذا العام 1442هـ .
 وقامت وكالة التشييد والصيانة ممثلةً في الادارة العامة للجسور والانفاق برفع كفاءة وجاهزية شبكة الانفاق بمكة المكرمة والمشاعر المقدسة البالغ عددها ٥٨ نفقاً للسيارات والمشاة بأطوال بلغت ( ٣٤٧٢٧ م ) من خلال تنفيذ الاعمال المدنية المتضمنة رفع كفاءة البناء الانشائي والمعماري ومعالجة الشروخ الخرسانية بمساحة ( ١٨٨٠٩٩٥ م ٢ ) و معالجة فواصل التمدد بطول ( ٢٧٦٠ م . ط ) ودهان الحوائط بمساحة ( ٢٤٠١٧م٢ ) الى جانب تنفيذ اعمال التشغيل المشتملة على التجهيزات الكهربائية والمكانيكية وتركيب المراوح النفاثة الجديدة بعدد ( ٧٠ ) مروحة و حساسات قياس جودة الهواء بعدد ( ٢١ ) حسّاساً وتثبيت نظام التحكم في المراوح بعدد ( ٩ ) اجهزة واستحداث منظومة المراقبة التلفزيونية بعدد ( ١٥٠ ) كاميرا ثابتة و ( ٤٠ ) كاميرا متحركة و ( ٣٥ ) كاميرا بانورامية و ( ١٣ ) جهازاً للإدارة والتسجيل و ( ٥٠ ) شاشة مراقبة، وتنفيذ اعمال الصيانة المتضمنة ( صيانة المولدات الكهربائية – مراوح التهوية – وحدات تكييف الهواء – تجهيزات الانذار ومكافحة الحريق ).
 كما اشتملت الاعمال على نظافة أنفاق المشاة بالمشاعر المقدسة البالغة ( ٦ ) أنفاق بأطوال (٢٠٠٠ م.ط ) ، وقد وضعت أمانة العاصمة المقدسة خطة تشغيلية متكاملة لتشغيل وصيانة الانفاق تضمنت تسخير عدد ( ١٠ ) فرق ميدانية لتنفيذ أعمال الصيانة على مدار الساعة مزودة بـ ( 130 ) فني وعامل و ( ٢٠ ) معدة مختلفة الاستخدام و ( ١٣ ) مشرفاً، وتأتي هذه الجهود المتواصلة بإشراف عام من معالي أمين العاصمة المقدسة المهندس محمد القويحص ومتابعة مستمرة من وكيل الامين للتشييد والصيانة المهندس زهير سقاط ، لتمكين حجاج بيت الله الحرام من اداء نسكهم بكل يسر وسهولة بمشيئة الله
