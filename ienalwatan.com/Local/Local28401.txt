ضبط ( 3489854 ) مخالفاً لأنظمة الإقامة والعمل وأمن الحدود في مناطق المملكة 

أسفرت الحملات الميدانية المشتركة التي انطلقت يوم الأربعاء الموافق 26 / 2 / 1439 هـ وحتى نهاية يوم الخميس 8 / 11 / 1440 هـ لتعقب وضبط مخالفي أنظمة الإقامة والعمل وأمن الحدود، التي تمت في كافة مناطق المملكة عن النتائج التالية : –
 أولاً : بلغ إجمالي المخالفين الذين تم ضبطهم بالحملات الميدانية الأمنية والمشتركة في جميع مناطق المملكة ( 3489854 ) مخالفاً، منهم ( 2721206 ) مخالفين لنظام الإقامة، و( 538862 ) مخالفاً لنظام العمل، و ( 229786 ) مخالفاً لنظام أمن الحدود.
 ثانياً : بلغ إجمالي من تم ضبطهم خلال محاولتهم التسلل عبر الحدود إلى داخل المملكة ( 58843 ) شخصاً، ( 47 % ) منهم يمنيو الجنسية، و ( 50 % ) إثيوبيو الجنسية، و ( 3 % ) جنسيات أخرى، كما تم ضبط ( 2558 ) شخصاً لمحاولتهم التسلل عبر الحدود إلى خارج المملكة.
 ثالثاً : بلغ إجمالي المتورطين في نقل وإيواء مخالفي أنظمة الإقامة والعمل وأمن الحدود والتستر عليهم الذين تم ضبطهم ( 3996 ) شخصاً.
 رابعاً : بلغ إجمالي المواطنين الذين تم إيقافهم لتورطهم في نقل أو إيواء وافدين مخالفين للأنظمة ( 1380 ) مواطناً، تم استكمال الإجراءات النظامية بحق ( 1343 ) منهم وإخلاء سبيلهم، وتستكمل الجهات المختصة تنفيذ الأنظمة بحق بقية الموقوفين وعددهم ( 37 ) سعودياً.
 خامساً : بلغ إجمالي من يتم إخضاعهم حالياً لإجراءات تنفيذ الأنظمة ( 12702 ) وافد مخالف، منهم ( 10866 ) رجُلاً، و ( 1836 ) امرأة.
 سادساً : تم إيقاع العقوبات الفورية بحق ( 496525 ) مخالفاً، وإحالة ( 450109 ) مخالفين لبعثاتهم الدبلوماسية للحصول على وثائق سفر، وإحالة ( 575775 ) مخالفاً لاستكمال حجوزات سفرهم، وترحيل ( 868065 ) مخالفاً
