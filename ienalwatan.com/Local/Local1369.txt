مركز حي العسيلة بمكة يعقد جلسته الأولى بدورته الثانية ويعتمد برامجه ولجانه 

عقد مساء الثلاثاء الموافق ١٤٤١/٧/٨ مركزحي العسيلة جلسته الاولى من الدورة الانتخابية الثانية للمركز بحضور رئيس مركز حي العسيلة الشيخ بندر بن سعيد السويهري ومدير المركز الاستاذ حسن السويهري وسكرتير المركز الاستاذ أحمد الزهراني وامين الصندوق الاستاذ هاني الانصاري وجميع الأعضاء المنتخبين ، وتم خلال اللقاء استعراض و مناقشة عدة محاور تهم مركز حي العسيلة ، وتم اعتماد توزيع مهام وأدوار الأعضاء في دورته الجديدة ، والتي تم الإتفاق على ان تكون على النحو التالي :
 الشيخ / بندر بن سعيد السويهري ( رئسياً للمجلس )
 الدكتور / طلق المسعودي ( نائبا للرئيس )
 الأستاذ / حسن السويهري ( مديراً للمركز )
 الأستاذ / هاني الانصاري ( امين الصندوق )
 الأستاذ / احمد الزهراني ( سكرتير المركز )
 وتم ايضاً اسناد اللجان على اعضاء مجلس الادارة وكانت على النحو التالي :
 الاستاذ / محمد الجعيد ( رئيساً للجنه الاجتماعية ومشرفاً على التدريب و التطوير بالمركز )
 الاستاذ / بندر المقاطي ( رئيساً للجنه الاعمال التطوعية )
 الأستاذ المحامي / عبدالرحمن الدوسري ( رئيساً للجنة اصلاح ذات البين )
 الاستاذ / عبدالله القرني ( رئيساً للجنة الانشطه والفعاليات )
 الاستاذ / سند الذيابي ( رئيساً للجنه الخدمات العامة )
 الاستاذ / احمد الهذلي ( رئيساً للجنه العلاقات العامة والاعلام )
 وفي ختام الإجتماع وقع جميع الأعضاء على ميثاق عمل مجلس الإدارة وبيان مهام كل عض
