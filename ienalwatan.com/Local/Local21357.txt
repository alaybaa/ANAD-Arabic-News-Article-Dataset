إعلان أسعار تعريفة الكهرباء الجديدة.. والتطبيق 1 يناير 

أعلنت هيئة تنظيم الكهرباء والإنتاج المزدوج، اليوم (الثلاثاء)، عن تعديل أسعار التعريفة الكهربائية، والتي ستدخل حيز التنفيذ ابتداء من مطلع شهر يناير 2018م، وذلك في إطار تصحيح أسعار منتجات الطاقة.
  
 وسيطال التعديل كلاً من: فئة الاستهلاك السكني، وفئة الاستهلاك التجاري، وفئة الاستهلاك الزراعي والمؤسسات والجمعيات الخيرية وما في حكمها، وفئة الاستهلاك للمنشآت الصحية الخاصة والمؤسسات والمعاهد والمدارس الأهلية.
  
 أما فئة الاستهلاك الصناعي وفئة الاستهلاك الحكومي فلن يطالهما أيّ تغيير، وذلك وفقاً للتفصيل التالي:
 شرائح الاستهلاك ( كيلو واط ساعة / شهر ) التعريفة ( هللة / كيلو واط ساعة )
  شرائح الاستهلاك ( كيلو واط ساعة / شهر ) التعريفة ( هللة / كيلو واط ساعة )
 – فئة الاستهلاك الزراعي والمؤسسات والجمعيات الخيرية، وما في حكم ذلك :
  شرائح الاستهلاك ( كيلو واط ساعة / شهر ) التعريفة ( هللة / كيلو واط ساعة )
 
 – فئة الاستهلاك للمنشآت الصحية الخاصة والمؤسسات والمعاهد والمدارس الأهلية :
  شرائح الاستهلاك ( كيلو واط ساعة / شهر ) التعريفة ( هللة / كيلو واط ساعة )
 والجدير بالذكر أنه لم يطرأ أي تغيير على تعريفة فئتي الاستهلاك الصناعي والحكومي :
  شرائح الاستهلاك ( كيلو واط ساعة / شهر ) التعريفة ( هللة / كيلو واط ساعة )
  شرائح الاستهلاك ( كيلو واط ساعة / شهر ) التعريفة ( هللة / كيلو واط ساعة )
 وأكدت الهيئة في بيانها “على أن هذه الخطوة تهدف إلى رفع الكفاءة الاقتصادية، ومساهمة القطاعات غير النفطية كمحرك تنموي اقتصادي مستدام، وترشيد استهلاك الموارد الطبيعية “منتجات الطاقة” واستدامتها للأجيال القادمة.”
 وبينت الهيئة أن التعريفة المعتمدة صممت لتغطي تكاليف الخدمة وضمان وفرة الكهرباء في جميع أرجاء المملكة، كما تساهم في تحسين جودة الخدمة وزيادة الإنتاجية وفتح مجال المنافسة وتنويع المصادر.
 كما أوضحت الهيئة أنه جاري التنسيق مع الشركة السعودية للكهرباء وشركة مرافق المياه والكهرباء بالجبيل وينبع لاتخاذ كافة الإجراءات اللازمة لتنفيذ ما جاء في قرار مجلس الوزراء، وتعديل نظام حساب الفواتير وفقاً لذلك
